All Hadith Books API

A static JSON API serving the 17 major Hadith books. Hosted on GitHub Pages.

Base URL

https://uthumany.github.io/All-Hadith-books/api

Endpoints

GET /books.json

Returns a list of all 17 books with metadata.

GET /{book-slug}.json

Returns the full collection of hadiths for a specific book.

GET /{book-slug}/hadith/{id}.json

Returns a single hadith by its ID.

GET /{book-slug}/page/{n}.json

Returns a paginated list of hadiths (50 per page).

GET /search.json

Returns a searchable index across all books.
